# 前后端联调检查清单（Day3后续）

## 1. 基础连通

- [ ] 前端 API_BASE 指向正确后端地址。
- [ ] 后端以 nifi 模式启动并可访问 `/api/v1/system/executor`。
- [ ] 返回 `mode=nifi` 且 `flowMappingSize=5`。

## 2. 任务链路（5类任务）

- [ ] CONVERT 创建成功并最终 `SUCCEEDED`。
- [ ] TAG_AUTO 创建成功并最终 `SUCCEEDED`。
- [ ] COPY_MULTI_FORMAT 创建成功并最终 `SUCCEEDED`。
- [ ] IMPORT_EXTERNAL 创建成功并最终 `SUCCEEDED`。
- [ ] TAG_MANUAL 创建成功并最终 `SUCCEEDED`。

## 3. 字段回填一致性

- [ ] 任务详情含 `nifiFlowId`。
- [ ] 任务详情含 `nifiRequestId`。
- [ ] 任务详情含 `nifiActiveThreadCount`。
- [ ] 任务详情含 `nifiRetryCount`。

## 4. 失败态一致性（后端错误码 -> 前端提示）

- [ ] `NIFI_AUTH_ERROR` 显示“鉴权失败”提示。
- [ ] `NIFI_NETWORK_ERROR` 显示“网络连接异常”提示。
- [ ] `NIFI_FLOW_NOT_FOUND` 显示“流程不存在”提示。
- [ ] `NIFI_FLOW_UNMAPPED` 显示“未配置流程映射”提示。
- [ ] `NIFI_EXEC_ERROR` 显示“执行异常”兜底提示。

## 5. 产物与文件中心

- [ ] `/api/v1/jobs/{jobId}/outputs` 返回产物列表。
- [ ] 文件中心可加载列表。
- [ ] 文件预览支持 CSV/JSON/TSV。
- [ ] 文件下载可用。

- [ ] 报告文件生成成功。
- [ ] 两轮任务总成功率 100%。
